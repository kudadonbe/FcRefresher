# Training Templates

This folder contains reusable templates for creating training materials.

## Available Templates

### Presentation Templates
- Marp presentation template with Family Court branding
- PowerPoint templates for various session types

### Document Templates
- Training handout template
- Practice exercise template
- Quiz/assessment template
- Feedback form template

## Using Templates

1. Copy the template you need
2. Rename following the format: `YYYY-MM-DD_TrainingName_Type.ext`
3. Update content while maintaining structure
4. Add Family Court metadata

## Template Standards

All templates should include:
- Family Court branding
- Presenter information placeholders
- Standard footer with contact details
- Consistent color scheme and fonts
