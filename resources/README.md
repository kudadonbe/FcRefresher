# Training Resources

Shared resources used across multiple training sessions.

### Images & Graphics
- Family Court logos and branding
- Icons and illustrations for presentations
- Screenshots and diagrams

### Reference Materials
- Quick reference cards
- Keyboard shortcut guides
- Best practices documentation
- Standard operating procedures

### Shared Documents
- Court case type codes

- Common troubleshooting guides

## Organization

Resources are organized by type:
- `images/` - Visual assets
- `guides/` - Step-by-step guides
- `references/` - Quick reference materials
- `branding/` - Official Family Court branding assets

## Usage Guidelines

- Always use official Family Court branding
- Maintain consistent visual style across trainings
- Credit sources for external materials
- Keep resources up-to-date
